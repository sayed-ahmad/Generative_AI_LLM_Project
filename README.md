# Generative AI Data Science Interview Chatbot

This project provides a lightweight, interactive chatbot for **Data Science interview preparation**.

## What it does

- Simulates Data Science interview questions across core topics (statistics, machine learning, SQL)
- Explains key concepts in interview-friendly language
- Supports interactive CLI learning with personalized focus topics

## Goal

Make interview preparation more **structured**, **interactive**, and **personalized** using modern AI-style chatbot workflows.

## Quick start

```bash
python chatbot.py
```

### Example commands

- `question`
- `question sql advanced`
- `explain overfitting`
- `focus: statistics, sql, machine learning`
